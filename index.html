Abstract

What LRCM does and why it matters

Current dance motion generation methods suffer from coarse semantic control and poor coherence in long sequences. We introduce LRCM (Listen to Rhythm, Choose Movements), a multimodal-guided diffusion framework that simultaneously leverages audio rhythm and hierarchical text descriptions (global style + local movements) for high-quality, controllable dance synthesis.

LRCM's key innovations include: (1) a Decoupled Multimodal Dance Dataset paradigm that separates motion capture data, audio rhythm, and text annotations for fine-grained semantic alignment; (2) a Heterogeneous Multimodal-Guided Diffusion Architecture with an Audio-latent Conformer extracting rhythmic cues and a Text-latent Cross-Conformer for semantic understanding; and (3) a Motion Temporal Mamba Module (MTMM) based on state space models for efficient autoregressive long-sequence generation.

Extensive experiments on 7 dance genres (Hip-hop, Krump, Popping, Locking, Jazz, Charleston, Tap) demonstrate that LRCM achieves state-of-the-art performance, improving FID by 35%+ and diversity by 42%+ over prior methods.

Method

Three core innovations that power LRCM

1

Decoupled Multimodal Dance Dataset

A feature-decoupling paradigm that separates motion capture data, audio rhythm features, and professionally annotated text descriptions (global style + local movement). Enables fine-grained semantic alignment between modalities across 7 dance genres.

Feature Decoupling7 Genres325 mins
2

Heterogeneous Multimodal-Guided Diffusion

Stacked denoising residual blocks with two specialized conformers: an Audio-latent Conformer for rhythmic cues and a Text-latent Cross-Conformer for fine-grained semantic understanding from global and local text.

Audio ConformerText Cross-ConformerDDPM Diffusion
3

Motion Temporal Mamba Module (MTMM)

State space model-based autoregressive extension enabling efficient long-sequence generation. MTMM concatenates previous segment memory with current latents and applies bidirectional Mamba scan for smooth temporal transitions.
